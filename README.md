# Jigar Parmar Portfolio

Futuristic AI/ML Engineer portfolio built with Next.js 15, TypeScript, Tailwind CSS, Framer Motion, and Three.js.

## Available scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run start`

## Notes

- GitHub section uses the public GitHub API to display repository cards.
- Theme preference is stored in `localStorage`.
- 3D neural network visualization uses `three` and renders inside the hero section.
